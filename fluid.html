About the project

Fluid is a customer relationship management platform that gives companies the ability to summon leads, engage customers, and promote a brand.

The application provides all the necessary marketing software on one easy-to-use platform.

The project is divided into three parts: a mobile app, a website for customers, and a platform for admin. It has many different functionalities, like campaigns, social media management, lead purchasing and an opportunity for live chat.

Admin Panel

The admin panel was created for product promotion and allows the user to add products to shareable, edit them, set up commissions, create availability regions, and perform various other functions such as promotions, giveaways, and others.

Settings

In the Settings section of the admin panel, the user is provided with the opportunity to specify the product's options, including color and size, specify the quantity of available product options, set a tax, set the cost of not only the product but also the delivery, edit and deactivate the product, in addition to the general description.

Messaging

Aside from the standard functions, Fluid offers a few handy options, like a quick messaging start with pre-made messages, a user can submit a payment request with a link, and a message can be sent to an email, like a text or SMS message in the app chat.

Also, the app has some special opportunities for the admin, for example, bulk messaging, where he can use templates with pre-built filters for audiences or create it from scratch. The channels could be created for communicating with the team and broadcast channels only for notifications without responding opportunities.

Payment request

Fluid has a convenient function for payment requests, such as generating a payment link. Here, the user can choose goods, payment purposes, and how long the link will be valid. There are additional filters and categories for goods for large companies.
