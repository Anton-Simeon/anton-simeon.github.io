About this project

The company’s slogan is “Simple Solutions for Difficult Tasks”. So the design also must be easy, especially in development, but it still must look stylish and contain only the necessary information.

The site was designed specifically for mobiles, along with a desktop design, ensuring that it looks good on all devices. It has some animated features as well.

Another feature of the project is that animations don't have an effect on the site’s speed, and it still loads content quickly.
